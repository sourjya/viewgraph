ViewGraph

The UI context layer for AI coding agents. Capture DOM snapshots, annotate issues, and send them to your AI agent for fixing.

Get started in 3 steps

  1. Connect to your agent

    Run from your project folder:

    npx @viewgraph/core init

    This configures the MCP server for Kiro, Claude Code, Cursor, or Windsurf.

  2. Capture a page

    Open your app in the browser and click the ViewGraph icon in the toolbar. The sidebar opens with Review and Inspect tabs.

  3. Annotate and send

    Click elements to annotate issues. Add comments describing what's wrong. Click Send to Agent and your AI agent receives the full DOM context.
