# @vg-help

List all ViewGraph MCP tools grouped by category. For each tool, show:
1. The tool name
2. A one-sentence plain-English explanation of what it does
3. When you'd use it
4. Example usage

Group them as: Discovery, Auditing, Annotations, Comparison, Agent-to-Browser, and Sessions.

Use `list_captures` to verify the server is connected before listing tools. If the server isn't connected, tell the user to run `viewgraph-init.js` first.
