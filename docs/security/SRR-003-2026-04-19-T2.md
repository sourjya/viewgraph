## Fixes Applied

### S5-6: classifyTrust hostname matching (MEDIUM → FIXED)
- **Before**: `pageUrl.includes(pattern)` matched against full URL including query params
- **After**: Parses URL, matches `hostname:port` only (same approach as S5-4)
- **File**: `extension/lib/constants.js`

### S8-1: Stdin close fallback timeout (MEDIUM → FIXED)
- **Before**: When idle timeout=0 and stdin closes, server persists forever
- **After**: `startStdinCloseFallback()` sets a 60-min fallback timer when idle timeout is disabled
- **File**: `server/index.js`
